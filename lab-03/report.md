**Цель работы:** Научиться создавать запросы с условием, в том числе с использованием логических операторов; научиться сортировать данные по возрастанию и убыванию.

## 1. Настройка среды разработки (Docker Compose)

Лабораторная работа выполняется на той же базе данных `student`, что и предыдущая. База запускается в изолированном контейнере через файл `docker-compose.yml` в папке `lab-03`.

```yaml
services:
  db:
    image: mysql:8.0
    container_name: mysql-lab03
    restart: always
    command:
      [
        "mysqld",
        "--character-set-server=utf8mb4",
        "--collation-server=utf8mb4_unicode_ci",
      ]
    environment:
      MYSQL_ROOT_PASSWORD: secret
      MYSQL_DATABASE: lab
    ports:
      - "3309:3306"
    volumes:
      - lab03-data:/var/lib/mysql
      - ./init.sql:/docker-entrypoint-initdb.d/init.sql
    networks:
      - shared
```

`ports: "3309:3306"` — каждая лабораторная использует уникальный порт хоста, что позволяет запускать несколько лабораторных одновременно без конфликтов.

`volumes: ./init.sql` — монтирует файл схемы напрямую. MySQL автоматически выполняет его при первом запуске контейнера, создавая все таблицы и заполняя их тестовыми данными.

## 2. Теоретические сведения

Предложение `WHERE` позволяет задавать условие фильтрации строк. В условиях можно использовать операторы сравнения `=`, `>`, `<`, `>=`, `<=`, `<>`, а также логические операторы `AND`, `OR`, `NOT`.

Оператор `GROUP BY` группирует записи по значениям одного или нескольких столбцов. Оператор `HAVING` применяется совместно с `GROUP BY` и задаёт фильтр для сформированных групп — в отличие от `WHERE`, который фильтрует отдельные строки.

Для работы с диапазонами значений используется `BETWEEN`, для проверки вхождения в список — `IN` и `NOT IN`, для поиска по шаблону — `LIKE` с символами `%` (любое количество символов) и `_` (один символ). Для проверки на пустое значение применяется `IS NULL`.

Сортировка результатов выполняется с помощью `ORDER BY`. По умолчанию сортировка производится по возрастанию (`ASC`), ключевое слово `DESC` задаёт обратный порядок.

### Задание 20. Выбрать студентов с номерами телефонов МТС, начинающимися с 3

Номера МТС начинаются с `8918` или `8919`. Шаблон `%3` проверяет, что третья цифра (после `891`) равна `3` — однако точнее использовать `LIKE '8918%'` в сочетании с дополнительным условием.

```sql
SELECT fam, telephone FROM dannie
WHERE (telephone LIKE '8918%' OR telephone LIKE '8919%')
  AND telephone LIKE '8918%3%';
```
